# AI-ML Journey

A beginner-friendly Python learning project that documents a step-by-step AI/ML study path. This repository contains daily exercise scripts, working examples of Python fundamentals, and a small CSV-based data analysis task.

## Project Overview

This project is a personal AI/ML learning journey focused on Python basics and introductory data processing. Each script explores core programming concepts and applies them to simple tasks that are useful for future AI/ML projects.

## Repository Structure

- `day1_exercise.py` - Python fundamentals: print statements, variables, type hints, lists, tuples, sets, dictionaries, and collection operations.
- `day2_exercise.py` - Random number generation, duplicate removal, sorting, range grouping, and finding the most common numeric range using functions with type hints.
- `day3_exercise.py` - List, dictionary, and set comprehensions; `*args`, `**kwargs`; enumerate examples; lambda functions; and basic machine learning-style function patterns.
- `day4_exercise.py` - CSV data processing for employee records: reading data, filtering by age, grouping by city, calculating average salary, sorting results, and exporting summaries to `results.csv`.
- `employees.csv` - Sample employee dataset used by `day4_exercise.py`.
- `results.csv` - Output generated by the data processing script.
- `sett.py` - (Not documented) project helper or settings file.

## What You Will Learn

- Python data types and collection behavior
- List, tuple, set, and dictionary operations
- Working with comprehensions for concise data transformation
- Function definitions with `*args`, `**kwargs`, and lambdas
- Basic file I/O and CSV processing
- Grouping and aggregating data by city
- Sorting and exporting results to CSV

## How to Run

1. Open a terminal in this folder.
2. Activate the virtual environment:
   - Windows: `venv\Scripts\activate`
3. Run any script with Python, for example:
   - `python day1_exercise.py`
   - `python day2_exercise.py`
   - `python day3_exercise.py`
   - `python day4_exercise.py`

## Notes

- `day4_exercise.py` reads `employees.csv` and writes `results.csv`.
- The scripts are designed as learning examples, not production-ready modules.
